# Bootstrap Hyperscript

A helper library for hyperscript that makes it easy to add bootstrap classes as components for any hyperscript renderer.

## Example

``` js
document.body.appendChild(
  container([
    navbar([
      brand(['Foo Company'])
    ]),
    jumbotron([
      h('h1', ['Welcome'])
    ])
  ])
)
```
